php mongodb扩展安装

发表于 2016-04-06 | | 阅读次数:

前言

由于项目中需要运用mongodb数据库，所以需要安装一下该扩展，在安装过程中有一些坑，故写一下安装流程以及该注意的一些东西。

安装前

首先安装扩展前，你需要安装mongodb，这里不赘述，然后你需要知道自己的php是什么样型号，笔者用的是wamp集成的环境，可以通过点击localhost-》phpinfo()中查看，笔者php情况如图，版本为5.4.3，64位，vc9。

下载安装

下载

知道了自己php的相关信息后，可以去官网站下载合适的mongodb扩展文件。
【注意】：根据的php版本下载对应的版本，以及选择正确x64还是x86，笔者当时看到php5.4的只有x86版本，心里吐了一口老血，什么鬼，竟然没有，在网上找了半天，终于找到合适的。分享给大家。【点击这里】

配置

下载完你需要的二进制包后，解压压缩包，将’php_mongo.dll’文件（下载过来的文件由于版本，可能名字有些不同，可以统一改成如上）添加到你的PHP扩展目录中（ext）。ext目录通常在PHP安装目录下的ext目录。
打开php配置文件 php.ini 添加以下配置：

extension=php_mongo.dll //注意不要在前面加“#”号

【注意】：由于笔者用的wamp，点开wamp，也可以进入到php.ini文件，但是通过wamp编辑该文件最终是不会生效的，所以应该到wamp安装目录下，找到bin-》php-》php.ini，两个文件都应该更改后才行。

修改好后，重启服务器。通过浏览器访问phpinfo，如果安装成功，就会看到类型以下的信息：

# php # mongodb
